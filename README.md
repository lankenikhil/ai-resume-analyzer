# AI Resume Analyzer
LIVE DEMO = [https://ai-resume-analyzer-beryl-chi-75.vercel.app/](url)

AI Resume Analyzer is a modern full-stack web application designed to help users evaluate and improve their resumes using AI-powered analysis and ATS (Applicant Tracking System) insights. The platform allows users to upload resumes, analyze resume content, identify missing skills, and receive smart recommendations to enhance job readiness and resume quality.

The application is built using Next.js, React, TypeScript, Tailwind CSS, and Prisma ORM, providing a responsive user interface and efficient backend integration. It also includes secure data handling, modern UI components, and optimized performance for a smooth user experience.

## Features

* Resume upload and analysis
* ATS-friendly resume evaluation
* AI-powered recommendations and insights
* Responsive and modern UI design
* Database integration using Prisma ORM
* Fast and scalable Next.js architecture
* GitHub and Vercel deployment support

## Tech Stack

* Next.js
* React.js
* TypeScript
* Tailwind CSS
* Prisma ORM
* SQLite / PostgreSQL
* Vercel

## Deployment

The project is deployed using Vercel and source code is managed with GitHub for version control and continuous deployment.
